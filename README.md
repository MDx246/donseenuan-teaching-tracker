# ระบบติดตามการสอนรายชั่วโมง โรงเรียนดอนสีนวน

เว็บแอปหน้าเดียวสำหรับติดตามหัวข้อการสอนรายชั่วโมงของโรงเรียนดอนสีนวน (สำนักงานสลากกินแบ่งสงเคราะห์ 523)

## ใช้งาน

เปิด `index.html` หรือเผยแพร่ผ่าน GitHub Pages ได้ทันที

## ความสามารถหลัก

- ติดตามหัวข้อรายชั่วโมงตามรายวิชา ชั้น และภาคเรียน
- แสดงความคืบหน้าการสอน
- ปุ่มสอนวันนี้
- ตารางสอนรายครูและคาบตามวันที่เลือก
- มุมมองตาราง ไทม์ไลน์ และตารางสัปดาห์
- รายงานสรุป
- นำเข้า/ส่งออก Excel, CSV และ JSON

## การบันทึกข้อมูล

ข้อมูลถูกบันทึกในเบราว์เซอร์ของผู้ใช้ด้วย `localStorage`
หากใช้งานจริงควรส่งออกไฟล์สำรองเป็นระยะ หรือเชื่อมต่อฐานข้อมูลกลางในอนาคต
